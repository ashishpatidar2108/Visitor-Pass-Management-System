# Frontend Source Code

The React frontend source code required for evaluation is included in this
repository under `frontend/src`.

## Important Files

- `frontend/src/App.jsx` defines the React routes.
- `frontend/src/main.jsx` mounts the React application.
- `frontend/src/pages` contains the page components for login, register,
  dashboard, visitors, appointments, passes, scan, reports, staff, database,
  OTP verification, forgot password, and visitor portal.
- `frontend/src/components/layout/AppLayout.jsx` contains the application
  layout and navigation.
- `frontend/src/components/routing` contains protected and role-based routing.
- `frontend/src/components/scanner/QrScanner.jsx` contains the QR scanner
  integration.
- `frontend/src/services/api.js` contains the frontend API client.
- `frontend/src/utils` contains auth storage and CSV export helpers.
- `frontend/src/styles/style.css` contains the main UI styling.

## Submission Note

If a submission portal does not show folders clearly, upload
`frontend-source-code.zip`. It contains the same React source files from
`frontend/src`.
